# Competitive Analysis

Three-phase pipeline: **scope → score → report**. Each phase feeds the next.

## When to Use

- Scoping a competitive landscape — who counts, which tier, where to look
- Scoring competitors across positioning, voice, craft, evidence, pricing, and strategic tension
- Assembling a decision-grade report with white-space analysis and recommendations
- Before a brand or positioning project to ground decisions in market evidence

## When NOT to Use

- One-off competitor lookups without a full pipeline — use research-ops instead
- Simple pricing comparisons — research-ops is lighter
- After the report is already written (use this before, not after)

## Quick Reference

| Phase | Input | Output | Est. effort |
|-------|-------|--------|-------------|
| 1 — Scope | Positioning brief | Tiered set (10-18 candidates), axis tags | 2-4 hrs |
| 2 — Score | Tiered set, sources | Profile cards per competitor, tension plot | 4-6 hrs |
| 3 — Report | Profile cards | Decision-grade report, alignment questions | 3-5 hrs |

## Key Principles (all phases)

- **No positioning brief = no analysis.** The client's lens determines what counts as a rival. Always establish it first.
- **No composite score.** Report dimension scores individually. A weighted average hides asymmetric strengths.
- **Asserted ≠ proven.** Site copy is marketing. Verify across ≥2 sources. Tag claims as asserted vs proven.
- **Every score needs evidence.** One-line justification + source link. No evidence = no score.
- **Tension axes never averaged.** Score both poles separately. The gap is the insight.
